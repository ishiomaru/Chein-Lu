# 自動化スクリプト (Eigo Sapuri Automation)

Playwrightを使用して、特定のWebアプリケーション（英語サプリ）の学習用画面の操作を自動化・補助するためのスクリプトです。

## 前提条件
- [Node.js](https://nodejs.org/) がインストールされていること。

## インストール手順

リポジトリをクローンまたはダウンロードし、プロジェクトのルートディレクトリで以下のコマンドを実行して依存関係をインストールします。

```bash
npm install
```

## 使い方

本ツールは「ログイン情報の保存」と「自動操作の実行」の2ステップに分かれています。

### ステップ1: ログイン情報の保存（`login.js`）

初回実行時やログインセッションが切れた場合は、まずブラウザでログインを行い、認証情報を `auth.json` に保存します。

```bash
node login.js
```

1. ブラウザが立ち上がり、対象のサイトが開きます。
2. 手動でログインを完了させてください。
3. ターミナルに戻り、`Enter` キーを押します。
4. `auth.json` が生成され、ブラウザが閉じます。

### ステップ2: 自動操作の実行（`practice.js`）

保存した `auth.json` を使用して、指定したURLでの自動操作を開始します。

1. `practice.js` 内の `URL` 変数を開きたい学習ページのURLに変更します。
   ```javascript
   const URL = "https://app.eigosapuri.jp/es/homeworks/...";
   ```
2. スクリプトを実行します。
   ```bash
   node practice.js
   ```
3. ブラウザが立ち上がり、指定したURLへ遷移します。
4. ターミナルに `準備完了。Enterキーを押すとループを開始します...` と表示されたら、ターミナルで `Enter` キーを押すと自動化が開始されます。

**自動化される操作:**
以下のボタンが表示された場合、自動でクリックして進行します。
- はじめる
- 正解をチェック
- できた
- 次へ (トレーニング)
- 次へ (リザルト)
- 次のレッスンへ

**終了方法:**
ターミナル上で `Ctrl + C` を押すと、安全にスクリプトを終了しブラウザを閉じます。

## ファイル構成
- `login.js` : 初回ログイン用スクリプト。手動ログイン後に認証情報を保存します。
- `practice.js` : メインの自動操作スクリプト。保存された認証情報を用いて操作をループ実行します。
- `package.json` : パッケージ設定ファイル（`playwright` を含む）。
